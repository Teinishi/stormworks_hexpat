# stormworks_hexpat

ImHex pattern definitions for Stormworks binary files

We are analyzing Stormworks-specific binary files and creating pattern files for [ImHex](https://github.com/WerWolv/ImHex). This project is not affiliated with Geometa and based on original research; its content may contain errors.

## Contents

- Pattern for .mesh file
- Pattern for .phys file

### WIP

- Pattern for .anim file
